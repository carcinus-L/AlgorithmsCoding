假设求最大平均通过率函数为g(n), 某个人数为i班级的通过率为 f(i),则：  
```g(n)=(f(i1) + f(i2) + f(i3) + ... + f(in)) / n```   
因为分母n是固定的, 所以分子增大, g(n)增大  

某个班级的通过认识为p,则:  
```f(i) = p / i```   
如果给该班级分配一个优秀生,那么  
```f(i + 1) = (p + 1) / (i + 1)```  
增长效果X为  
```
X = f(i + 1) - f(i)
X = (p + 1) / (i + 1) - p / i
X = (i * (p + 1) - p * ( i + 1)) / i * (i + 1)
X = (i * p + i - p * i - p) / i * (i + 1)
X = i - p / i * (i + 1)
```
所以每次找最大的X即可
